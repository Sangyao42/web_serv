Webserv Tests for Evaluation

Test 01

Main Page

Do: Send GET Request of /

Expected: 200 OK

Test 02

Folder with credentials

Do: Send GET Request of /usernames

Expected: 403 Forbidden (chmod -r www/usernames) / 405 Method Not Allowed (chmod +r www/usernames)

Test 03

Directory Listing

Do: Send GET Request of /display (autoindex off)

Expected: 200 OK, showing index.html of the /display/

Do: Click delete button for sample texts, send DELETE request of /upload/[filename.txt]

Expected: 204 No Content (successfully deleted)

Test 04-1

File Upload

Do: Send POST Request for file upload (plain, txt, html, jpg, jpeg, png)

Expected: 201 Created, check if the file is in the /upload/

Test 04-2

File Upload

Do: Send POST Request for file upload (same file, modification)

Expected: 200 OK

Test 04-3

File Upload

Test 04-4

File Upload
